Synopsis

Zombie Hunter is a 2D top-down shooter with a hint of adventure. You start out as the hero with a mission to kill and reclaim as many souls as possible from the zombie horde that remain after a major catastrophe. Armed with just a pistol to begin with you must trawl your way through the waves of undead searching for the keys of light. On your travels you will find better weapons to help you progress and once you have at least four keys and reclaimed 200 souls you wil be granted the right to ascend to the next dimension.

Development

Developed for both Desktop and Mobile platforms, designed to be responsive, as is this site I created to host the game. Based in JavaScript using the Phaser 2 API for a university project. Most sprites were created using Piskel https://www.piskelapp.com/ and the environment was created from a free to use tile set using Tiled https://www.mapeditor.org/. The virtual keyboard for highscore name entry was developed using the GameMaker 2 engines sprite editor. All other visual tweaks and edits were performed with Adobe Photoshop. Sounds were all separately sourced from https://freesound.org.
